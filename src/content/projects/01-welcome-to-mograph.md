# Welcome to Monograph
This is your portfolio index. Each entry you see here is a project stored in `src/content/projects/`.


## Quick Start
1. **Delete the demo projects** (this file and others starting with `00-`, `01-`, etc.)
2. **Add your own projects** by creating new `.md` files in `src/content/projects/`
3. **Customize the homepage bio** by editing `src/content/pages/home.md`
4. **Update your CV** in `src/content/pages/cv.md`.


## What You're Looking At
- **Left Column:** Project index (you are here)
- **Center Column:** Main content (this text)
- **Right Column:** Your CV/contact info.
